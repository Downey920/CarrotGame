# CarrotGame
HTML CSS JavaScript
기본적으로 게임 시작 버튼을 누르면 벌레와 당근이 랜덤한 위치에 뿌려진다.
시간 내에 벌레를 피하여 당근을 모두 클릭 하면 이긴다.
